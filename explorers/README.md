# Allowed Explorers on the Spectra IBC app

This file contains the list of allowed explorers on the Spectra IBC app.

## List of Explorers

- Mintscan <https://mintscan.io>
- Spectra <https://thespectra.io>
- PingPub <https://ping.pub>
- PingPub Cogwheel <https://explorer.cogwheel.zone>

## How to add a new explorer

To add a new explorer, you need to submit a PR to the `explorers/README.md` file and add it to the `explorers/allowed_explorers.toml` file.
